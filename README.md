# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 폴더를 로컬에 준비합니다.
2. `index.html` 파일을 더블 클릭하거나, 브라우저 주소창에 파일 경로를 입력해 엽니다.

   예시 (Windows):

   ```
   file:///C:/DEV/tetris-cursor/index.html
   ```

3. **시작** 버튼을 누른 뒤 블록이 내려오고 키보드 조작이 가능하면 정상입니다.

> **빈 화면만 보일 때:** `index.html`을 더블 클릭해 `file://`로 열어야 합니다.
